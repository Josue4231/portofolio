Project realisé

Introduction

Ces deux projets réalisés dans un cadre pédagogique mettent en pratique mes compétences en développement web : l’un consiste à créer une interface interactive de découverte géographique avec PHP et SQL (Geoworld), et l’autre exploite une API REST pour afficher dynamiquement des données cinématographiques issues de la base TMDb (API Movie DB).

Geoworld

Date : Mars 2025

Technologies utilisées : PHP, SQL, HTML, CSS, JavaScript

Description : Geoworld est une application web permettant de visualiser et d'explorer des cartes interactives de différents continents et pays. Ce projet a été développé pour offrir une interface simple et interactive pour explorer les informations géographiques et historiques des pays à travers une architecture PHP/SQL.

Voir le projet geoworld

Projet : API Movie DB

Description :

Ce projet consiste à exploiter l’API REST de The Movie Database (TMDb) pour créer une interface web qui permet de consulter les films les plus populaires, les mieux notés, ou encore de rechercher des films ou acteurs. Les données sont récupérées via des appels HTTP et affichées dynamiquement en PHP.

Fonctionnalités développées :

  • Affichage des films populaires et mieux notés
  • Recherche de films par genre ou acteur
  • Consommation d'une API REST avec JSON

Technologies utilisées :

  • PHP
  • HTML / CSS
  • Bootstrap (pour le design)
  • API REST (TMDb)

Voir le projet API Movie DB
